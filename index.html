üìä How to Create and use in Google Sheets

1 Create the basic structure:
Open Google Sheets and create a new spreadsheet. In the first row, add the headers: A1: Item, B1: Label, C1: Category, D1: Owner
2 Create dropdown for Category (column C):
Select column C (from C2), go to Data > Data validation. Choose "List of items" and add: Electronics, Clothes, Books, Kitchen, Decoration, Documents, Toys, Tools, Furniture, Others
3 Create dropdown for Owner (column D):
Select column D (from D2), go to Data > Data validation. Choose "List of items" and add the residents' names (ex: John, Mary, Peter, Anna, Family)
4 Create search area:
In a separate area (ex: column F), create:
  • F1: "Search Item:"
  • G1: Cell to type the item
  • F2: "Label:"
  • F3: "Category:"
  • F4: "Owner:"
5 Add search formulas:
In the cells next to the search labels, add the formulas:
üîß Formulas to copy and paste:
In G2 (Label):
=IF(ISBLANK(G1),"",IFERROR(INDEX(B:B,MATCH(G1,A:A,0)),"Item not found"))
In G3 (Category):
=IF(ISBLANK(G1),"",IFERROR(INDEX(C:C,MATCH(G1,A:A,0)),"Item not found"))
In G4 (Owner):
=IF(ISBLANK(G1),"",IFERROR(INDEX(D:D,MATCH(G1,A:A,0)),"Item not found"))
